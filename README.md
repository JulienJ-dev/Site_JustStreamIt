# JustStreamIt

## Description

Interface web realisee dans le cadre de la formation OpenClassrooms. Le site affiche des films classes par categories a partir d'une API locale et permet de consulter les details d'un film dans une modale.

## Fonctionnalites

- Affichage du meilleur film
- Affichage des films les mieux notes
- Sections par categories de films
- Selection dynamique d'une categorie
- Recuperation des donnees depuis une API REST locale
- Affichage des details d'un film dans une fenetre modale
- Mise en page responsive en HTML, CSS et JavaScript

## Technologies utilisees

- HTML
- CSS
- JavaScript
- API OCMovies locale

## Lancer le projet

1. Lancer l'API OCMovies en local sur le port 8000.
2. Ouvrir le fichier `index.html` dans un navigateur.

L'application utilise l'URL suivante pour consommer l'API :

```text
http://localhost:8000/api/v1
```

## Fichiers principaux

- `index.html` : structure de la page
- `styles.css` : mise en forme et responsive design
- `script.js` : appels API, affichage dynamique et modales
- `Ressources/` : images et assets du projet

## Objectif portfolio

Ce projet montre ma capacite a construire une interface web dynamique, a consommer une API et a organiser du JavaScript cote client.
